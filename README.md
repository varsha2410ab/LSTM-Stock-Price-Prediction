# Stock Price Prediction using LSTM

This project uses a Long Short-Term Memory (LSTM) neural network to predict stock prices using data from Yahoo Finance.

## Features
- Data collection via `yfinance`
- Preprocessing with sliding windows and scaling
- LSTM neural network model
- Evaluation using RMSE, MAE, MAPE, and Directional Accuracy
- Visualization of predicted vs actual prices

## Usage
```bash
pip install -r requirements.txt
python main.py
```
